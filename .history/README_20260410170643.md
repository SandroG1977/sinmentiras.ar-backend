# sinmentiras.ar backend (FastAPI + LangChain + LangGraph)

## Setup

1. Activate your virtual environment.
2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Create your env file:

```bash
cp .env.example .env
```

4. Run the API:

```bash
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

## Endpoints

- `GET /` root check
- `GET /api/v1/health` health check
- `POST /api/v1/agent/chat` run LangGraph flow
- `POST /api/v1/rag/ingest/minio` ingest a MinIO file into RAG chunks DB

### Chat body example

```json
{
  "prompt": "Hola, dame un resumen del proyecto"
}
```

If `OPENAI_API_KEY` is empty, the agent returns a mock response.

## Local RAG (baseline)

This backend now includes a lightweight local RAG retriever.

1. Add knowledge files in `app/knowledge` (`.md`, `.txt`, `.json`).
2. Configure optional env vars:

```bash
RAG_ENABLED=true
RAG_TOP_K=3
RAG_KNOWLEDGE_PATHS=app/knowledge
```

Notes:
- `RAG_KNOWLEDGE_PATHS` can be a comma-separated list of files and/or directories.
- Retrieved chunks are injected into the model prompt as context before generation.
- Retrieval uses FAISS semantic search (with lexical fallback if FAISS is unavailable).

## Query cache (anti-repeat)

The `/api/v1/agent/chat` endpoint is used as a search query endpoint.
Before calling the LLM, the backend tries to find a semantically similar previous question
in the question cache DB.

Flow:
1. Receive query (`prompt`).
2. Embed the query.
3. Search question cache vectors.
4. If similarity >= `CACHE_MIN_SIMILARITY`, return cached answer.
5. Otherwise run RAG + LLM and store the new pair in cache.

Config:
- `CACHE_ENABLED`
- `CACHE_MIN_SIMILARITY`
- `CACHE_SQLITE_PATH`

### MinIO ingestion for RAG

Use this endpoint to ingest a text file from MinIO, chunk it, store chunks in SQLite,
and make it immediately available for retrieval.

`POST /api/v1/rag/ingest/minio`

Request example:

```json
{
  "minio_path": "minio://my-bucket/contexto/app_contexto.txt",
  "chunk_size": 700,
  "chunk_overlap": 120,
  "replace_existing": true
}
```

Response example:

```json
{
  "document_id": "fca62f...",
  "source_uri": "minio://my-bucket/contexto/app_contexto.txt",
  "chunks_inserted": 42,
  "bytes_read": 125430,
  "db_path": "app/data/rag.sqlite3"
}
```

Required env vars for MinIO access:
- `MINIO_ENDPOINT`
- `MINIO_ACCESS_KEY`
- `MINIO_SECRET_KEY`
- `MINIO_SECURE`

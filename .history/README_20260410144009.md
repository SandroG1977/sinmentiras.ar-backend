# sinmentiras.ar backend (FastAPI + LangChain + LangGraph)

## Setup

1. Activate your virtual environment.
2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Create your env file:

```bash
cp .env.example .env
```

4. Run the API:

```bash
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

## Endpoints

- `GET /` root check
- `GET /api/v1/health` health check
- `POST /api/v1/agent/chat` run LangGraph flow

### Chat body example

```json
{
  "prompt": "Hola, dame un resumen del proyecto"
}
```

If `OPENAI_API_KEY` is empty, the agent returns a mock response.
